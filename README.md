# Run Locally

Clone the project
```sh
git clone https://github.com/Aselivm/Hangman_rust.git
```
```sh
cd Hangman_rust
```
### Make sure you have installed Rust & Cargo - https://www.rust-lang.org/tools/install

```sh
rustc --version
```
```sh
cargo --version
```
### Compile and run the program with Cargo
```sh
cargo run
```
